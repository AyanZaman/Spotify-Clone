# 🎵 Spotify Web Player Clone

A frontend user interface clone of the popular music streaming platform, Spotify. This project was built to demonstrate proficiency in responsive web design, complex CSS layouts, and audio DOM manipulation using vanilla JavaScript.

🔴 **[Live Demo: Click Here to View](https://ayanzaman.github.io/Spotify-Clone/)**

## 🚀 Tech Stack
* **HTML5:** Semantic markup and structure.
* **CSS3:** Custom styling, flexbox layouts, hover effects, and responsive design.
* **JavaScript (Vanilla):** Dynamic audio playback controls, play/pause toggling, and track updating.

## 💡 Key Features
* Custom audio player with working play, pause, and skip functionality.
* Real-time progress bar tracking for the currently playing song.
* UI styling and hover animations that closely mimic the official Spotify desktop application.
